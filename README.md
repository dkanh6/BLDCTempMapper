**Simulation tool for hobbyists, researchers, etc., providing dynamic temperature estimates.**

This repository is a work in progress, but the plots below show both experimentally recorded results alongside results provided by a two-compartment equivalent circuit model. 

In the experimental plot, each differently colored segment indicates a change in the current used:
- <span style="color:red;">Red</span>: 0.24A
- <span style="color:green;">Green</span>: 1.31A
- <span style="color:blue;">Blue</span>: 0.33A
- <span style="color:aqua;">Aqua</span>: 0.68A
- <span style="color:pink;">Pink</span>: 1.35A
